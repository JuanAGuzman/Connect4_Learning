Para iniciar se debe ejecutar el archivo main, este despliega la interfaz grafica del juego.
En la pantalla de menu se tienen 4 opciones:

- PLAY: Si se selecciona se dirige a una pantalla para la seleccion de jugadores, con la seleccion lista se habilita el boton NEXT para proceder al juego,
para seleccionar la columna se hace click izquierdo sobre la columna deseada en cualquier fila. Al finalizar la ronda se anuncia el ganador o si es empate dependiendo
del caso. En la pantalla final se tienen las opciones para ir al menu, volver a jugar (se retorna a la seleccion de jugadores) y QUIT para salir del juego.

- QTable: Al seleccionar esta opcion se dirige a una pantalla de juego. Al lado izquierdo se encuentran botones de los dos modelos para seleccionar al que se desee
observar los proximos valores. El objetivo de esta escena es jugar individualmente para crear estados y a medida que se progresa se despliega el valor de recompensa
para cada accion en dicho estado. Se debe seguir hasta conseguir un empate o un jugador, nuevamente se despliega una pantalla que permite ir al menu, probar nuevamente
o salir. Esta la opcion de exportar un archivo csv para observar una lista de las tuplas estado, accion y recompensa. Es necesario seleccionar un agente para exportar
exitosamente.

- CREDITS: Esta pantalla dirige a una escena donde se da el nombre de la persona desarrolladora del juego. Se puede coninuar dando click en cualquier parte de la
pantalla para redirigir al menu.

- QUIT: Finalmente se tiene la opcion de salir del juego.
